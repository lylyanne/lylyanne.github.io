Understanding border, padding, and margin

16 August 2014

Let's say we have a div with background color of orange like below:

div {
	background-color: #FFA500;
}

We use border to put visual frame around our div.

Border has a few properties such as: border-width, border-style, and border-color.

  • border-width determines how thick the border is.
  • border-style defines the style of the border. Some common styles for borders are solid, dashed, and ridge. You can refer to W3schools for all possible styles.
  • border-color defines the color of the border, just like the name implied.

You can define a border using shorthand like below: 

	border: 5px solid black;

Padding

Padding is used to add additional space around our content inside the border. You can think of padding as fattening up the content area.

There are a couple ways you can specify padding.

	padding-top: 5px;	
	padding-right: 10px;
	padding-bottom: 5px;
	padding-left: 10px;

or

	padding: 5px 10px;

or

	padding: 5px;

The first and second way are equivalent to each other. When there are only 2 values specified such as what you see on the second example above, the first number (5px) refers to the top and bottom padding while the second number (10px) refers to the left and right padding.

If all four sides are supposed to be padded using the same value, then you can use the shorthand that you see on the third example.

Notice how padding makes our content a bit fatter?


Margin

Margin is used to create distance between our content to other content around it.

Margin can be defined the same way as padding.

	margin-top: 5px;	
	margin-right: 10px;
	margin-bottom: 5px;
	margin-left: 10px;

or

	margin: 5px 10px;

or

	margin: 10px;

Using the third example, this is how our example looks like next to other elements:

Compare it to what our div looks like without the margin.


In Summary

Use border to frame your content.

Use margin to distance content from other content.

Use padding to make your content a bit larger.
